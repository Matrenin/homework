	Репозиторий - хранилище проекта. Репозиторий отличается от обычной папки тем, что он хранит все слепки изменений.

ls - список файлов.

git log - получаем информацию о каждом коммите, который был создан в вашем репозитории.
git log --oneline - получаем историю о коммитах, но только сжатую.

	Создание репозитория:
cd name - Переходим в папку где будем создавать репозиторий. Где name - название папки. 
git init - Команда создает пустой репозиторий, в текущем месторасположении.
git status - показывает текущеее состояние нашего репозитория, какие файлы были добавлены, изменены или удалены.

	Создадим синхронизацию между локальными файлами и сервисом github:
git remote add origin link - Где link - ссылка репозитория, которую мы получаем на сервере github.
git remote -v - Команда показывает списик подключенных remotu соединений. Убеждаемся, что соединение прошло успешно.

	Создаем коммит:
Добавляем файл в папку репозитория.
git comit -am 'message' - Команда автоматически добавляет все изменения в индекс и оставляет коммит. Где message - сообщение к коммиту.

	Работа с ветками репозитория:
git branch - просмотр доступных веток репозитория.
git branch name - создаем новую ветку, где name - название ветки.
git checkout name - переключаемся на ветку name, где name - название ветки.
git branch newName name - создаем новую ветку. Name - название ветки от которой хотим создать копию. Где newName - назваине новой ветки.
git checkout -b test- Создаем новую ветку и переключаемся на нее. Test - название новой ветки.

	Настройка SSH подключение, для выполнения синхронизации:
ssh-keygen - генерируем ключ.
Ключ находится в файле по адресу ../.ssh/id_rsa.pub
cat link - выводим содержимое файла. Link - адрес где находится ключ. Содержимое файла мы копируем и вставляем в github, в настройки SSH.
git clone link name - копируем репозиторий на локальный компьютер. Link - маршрут откуда мы будем копировать репозиторий. Name - Название папки, куда будет скопирован репозиторий.

	Публикация репозитория:
git push -u origin name - загружаем данные в удаленный репозиторий и создаем ветку. Name - название ветки.
git pull - скачиваем изменения с удаленного репозитория на локальный.
