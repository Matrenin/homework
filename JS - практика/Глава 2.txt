	getComputedStyle(elem).style;
> Метод возвращает значения свойства style у элемента elem.

	array.join(separation);
> Объединяет все элименты массива array в строку. Separation - определяет строку, разделяющую элименты массива.

	array.sort(function);
> На месте сортуриет элементы массива array по указанной функции function, определяющую порядок сортировки.

	array.from(string, mapFn);
> Создает новый экземляр Array из массивоподобного или итерируемого объекта string. Mapfn - вызываемая функция для каждого элемента массива.

	element.closest(selector);
> Возвращает ближайший родительский элемент (или сам элемент), который соотсветсвует заданному CSS-селектору selector.


	Object.values(obj);
> Возвращает массив значений перечисляемых свойст объекта(obj) в том же порядке. Разница между циклом и мотодом в том, что цикл перечисляет свойства и из цепочки прототипов.

	array.reduce((acc, product) => acc + product.count, 0);
> Метод применяет функцию к каждому элементу массива, возвращая одно результирующее значение. Acc - аккумулятор, то что сохраняется между каждыми вызовами этой функции. Изначально acc = 0. Product.count - текущий обрабатываемый элемент массива.

	node.parentElement;
> Возвращает родителя узла node.

	element.previousElementSibling;
> Возвращает элемент стоящий перед применяемым(element), из списка дочерних элементов родителя.

	Object.assign(target, ... sources);
> Используется для копирования значений всех собственных перечисляемых свойст из одного или более исходных объектов(sources) в целевой объект(target).

	string.startsWith(searchString);
> Определяет, начинается ли стора с символа указанных в скобках(searchString). String - проверяемая строка.

	array.includes(searchElement);
> Определяет, содержит ли массив(array) определенный элемент(searchElement).
