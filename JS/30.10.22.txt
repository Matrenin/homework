	Params игнорируются, если указан path, что не является в случаем с query. Вместо этого, вам нужно указать name маршрута или вручную указать весь path с необходимыми параметрами.
const userId = '123'
router.push({ name: 'user', params: { userId } }) // -> /user/123
router.push({ path: `/user/${userId}` }) // -> /user/123
