	Type:
type="password" - для скрытого ввода.

type="radio" name="text" value="phone" - для создания кнопки переключателя, "name" - значение указывается  для всех кнопок одиннаковым, для выбора одной из них. "Value" - отправляет на сервер выбранное значение пользователя, если не указать значение value, то данные отправленные на сервер, будут бесполезными.
