	Препроцессоры - это программа, добавляющая новые возможности написания CSS кода. Они позволяют использовать при написании стилий свойственные языкам программирования приемы и конструкции: переменные, циклы, функции, вложенность.

	Переменныме - способ хранения информации, которую вы  хотите использовать при написании всех стилей проекта. Чтобы создать переменную в SCSS используется символ $ далее название переменной и ее значение ($color: #fff;).

	Фрагментирование. Вы можете создавать фрагменты SCSS-файла, содержащие в себе небольшие отрывки CSS, которые можно использовать в других SCSS-файлах. Фрагмент - это простой SCSS-файл имя которого начинается с нижнего подчеркивания. Нижнее подчеркивание говорит компилятору о том, что это только фрагмент и он ен должен компилироваться в CSS (_header.scss).

	Чтобы ссылка открывалась в новой вкладке тегу <a> добавляется атрибут target со значением "_blank". Если мы хотим, чтобы все ссылки открывались в новом окне, то в <head> прописываем команду <base target="_blank">.
