	Цикл — это повторяющаяся последовательность действий. Цикл состоит из условия и тела цикла. Перед запуском цикла, проверяется условие. Если условие истинное, то выполняется блок кода, который называется телом цикла. Затем этот шаг повторяется. Так будет продолжаться, пока условие не станет ложным. Каждое выполнение тела цикла называется итерацией.

	Массив — это структура, в которой можно хранить коллекции элементов. Массивы хранят элементы в пронумерованных «ячейках». Нумерация начинается с нуля. Первый элемент массива будет иметь номер 0, второй — 1 и так далее. Номера называют индексами.

	Number.isInteger() - метод определяет, является ли переданное значение целым числом.

	let lastElem = array.pop() - при удаления элемента из массива, его можно сохранить в переменную.

	Объект нужен для описания кого-то или что-то (пользователя, технику). Массивы нужны для сборки однотипных значений одной тематики. 

	------------
	
	Знак "?" это оператор опциональной последовательности. Он проверяет существует имя свойства в объекте, которое находится перед  оператором.

let user = {
	name: "Tagir",
	socialNetworks: {
		vk: "vk.com",
		facebook: "facebook.com",
  	}
};

console.log(user.socialNetworks?.facebook);

	Два объекта с одинаковыми значениями никогда не будут равны, потому что мы сравниваем ссылки в оперативной памяти.


let btn = document.querySelector("button");
let inputs = document.querySelectorAll("input");
let word = "cat";
word = word.split("");

btn.addEventListener("click", function() {
  for (let i = 0; i < word.length; i++) {
    let answer = inputs[i].value;
    if (word.includes(answer)) {
      inputs[i].style.backgroundColor = "yellow";
    }
    if (word[i] == answer) {
      inputs[i].style.backgroundColor = "green";
    }
  }
});
